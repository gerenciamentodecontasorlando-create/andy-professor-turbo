# ANDY — Assistente de Prática Clínica Integrada (PWA)

## O que é
Um app offline-first para preceptores: cadastro de alunos, avaliação prática (0/3/5), feedback pedagógico,
caso clínico do dia (anonimizado), relatórios e PDFs.

## Como usar (GitHub Pages)
1. Suba esta pasta como repositório.
2. Em Settings → Pages, selecione `Deploy from a branch` e `main / (root)`.
3. Acesse o link do Pages. No celular, clique em **Instalar**.

## Offline
O app funciona offline após o primeiro acesso (cache do app shell).
PDFs usam impressão para "Salvar como PDF".

## Backup
Use o botão **Backup (.json)** para exportar seus dados e importar em outro aparelho.

## Observação ética
O bloco “Caso Clínico” é um registro acadêmico anonimizado.
Não inclua dados identificáveis do paciente.
